Casabourse

library(casabourse)

Casabourse est un R package qui permet dobtenir des donnes temps rel de la bourse de Casablanca. Lobjectif est de faciliter laccs aux donnes tous les utilisateurs du langage de programmation R. Ce package comporte une diversit de donnes accessibles juste par appel de fonction.

Guide dinstallation

Ce package est disponible sur github et sur le CRAN pour linstaller vous devez excuter une de ces lignes de codes :

#CRAN version
install.packages("casabourse")
#Or github dev version
devtools::install_github("AODiakite/casabourse")

La fonction tickers()

Cette fonction ne prend aucun argument et retourne une table de donnes associant chaque entreprise un ticker. Il est important de visualiser les tickers. En effet, ils sont utiliss par les autres fonctions du package pour reprsenter les entreprises auxquelles ils correspondent.

La fonction daily.data(ticker, from, to) :

Comme son nom lindique, cette fonction renvoie les cours journaliers dun titre entre deux dates donnes par lutilisateur. Elle reoit trois arguments : * ticker : qui correspond au ticker du titre qui nous intresse ou mme de lindice MASI( pour rappelle vous pouvez connaitre tous les tickers des titres grce la fonction tickers() ) * from : reprsente la date de dbut de collecte de donnes. Cet argument est du type jour-mois-anne. * to : reprsente la date darrt de collecte de donnes. Cet argument est du type jour-mois-anne. NB : Les week-end ne sont pas pris en compte lors du tlchargement des donnes et en plus Il est important de mettre chaque argument entre des doubles quotes( )

#Affichage des cours journaliers de ATTIJARIWAFA BANK entre 01 janvier 2017 et le 14 decembre 2021
atw <- daily.data("atw","01-01-2017","14-12-2021") #affection de la table a la variable atw

Exemples dutilisations de la library(casabourse) :

  • Nous allons tracer la courbe de variations des cours des titres de Attijariwafa Bank et de la Banque Populaire du Maroc entre le dbut de lanne 2020 et le 15 dcembre 2021.
#lecture des donnes de Attijariwafa Bank
atw <- daily.data(ticker = "ATW", from = "01-01-2020", to = "15-12-2021")
#lecture des donnes de la Banque Populaire du Maroc
bcp <- daily.data(ticker = "BCP", from = "01-01-2020", to = "15-12-2021")
data=merge(atw,bcp, by = "Date")

plot_ATW =  ggplot(data = data, aes(x = Date,y = ATW))+
  geom_line()+
  labs(title = "Attijariwafa Bank")+
  ggthemes::theme_calc()# Pour changer de theme

plot_BCP=  ggplot(data = data, aes(x = Date,y = BCP))+
  geom_line()+
  labs(title = "Banque populaire")+
  ggthemes::theme_calc()# Pour changer de theme

library(patchwork)
plot_ATW|plot_BCP

  • Traons maintenant la courbe des taux de rentabilits des titres de Attijariwafa Bank et de la Banque Populaire du Maroc entre le 15 octobre 2021 et le 15 dcembre 2021.
rent <- function(x){
  returns = (x[-1]-x[-length(x)])/x[-length(x)]
  c(0,round(returns*100,2))
}
#lecture des donnes de Attijariwafa Bank
atw <- daily.data(ticker = "ATW", from = "15-10-2021", to = "15-12-2021")
#lecture des donnes de la Banque Populaire du Maroc
bcp <- daily.data(ticker = "BCP", from = "15-10-2021", to = "15-12-2021")
#Utilisation de la function rent
atw$returnATW = rent(atw$ATW)
bcp$returnBCP = rent(bcp$BCP)


#Traage de la courbe avec le package ggplot2
data = merge(atw,bcp,by = "Date")
plot_ATW =  ggplot(data = data, aes(x = Date,y = returnATW))+
  geom_line()+
  labs(title = "Rentabilit Attijariwafa Bank en %")+
  ggthemes::theme_calc()# Pour changer de theme

plot_BCP=  ggplot(data = data, aes(x = Date,y = returnBCP))+
  geom_line()+
  labs(title = "Rentabilit Banque populaire en %")+
  ggthemes::theme_calc()# Pour changer de theme

library(patchwork)
plot_ATW|plot_BCP
